# nerfdiff
NeRF Based Localization

# Setup
```
git clone https://github.com/naamiinepal/nerfdiff.git
cd nerfdiff
echo export PYTHONPATH=:\$PYTHONPATH:`pwd` >> ~/.bashrc
source ~/.bashrc

conda env create -f environment.yml 
conda activate nerfdiff
```
